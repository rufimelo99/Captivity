-----------BUILDING INSTRUCTIONS-----------

Follow the standard project building path as in any other Unity Project.

To do this in the Unity Editor, go to Assets > Import Package, and select the package you want to import. 
Select Create Project when you're done. 
Unity automatically generates the required files and creates your Project.

-----------RUNNING INSTRUCTIONS-----------

There is no need for any kind of installation.
To run the game, simply double-click the "Prototype.exe" file.
